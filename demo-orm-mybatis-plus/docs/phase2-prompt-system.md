# Phase2 - Prompt与用户画像工程

## 一、当前阶段目标

第一阶段已经完成：

- DeepSeek API
- Redis聊天上下文缓存
- SSE流式输出
- MySQL聊天记录
- 多轮对话

第二阶段目标：

1. Session会话系统
2. Prompt模板系统
3. Token统计
4. 用户画像系统
5. 长期记忆摘要
6. 上下文裁剪
7. AI配置中心

---

# 二、当前系统问题

当前AI系统：

- Prompt写死
- 一个用户仅一个聊天
- 无用户画像
- 无长期记忆
- 无Token统计
- AI回复缺乏个性化

导致：

- AI人格不稳定
- AI无法“记住用户”
- 无法形成真实聊天体验
- 无法支持后续AI陪伴能力

---

第二阶段重点：

```text
让AI开始“认识用户”
```
